# Steel Fault Detection Project

A comprehensive machine learning project for steel plate fault detection, consisting of three main components:

1. **Data Mining** - Data exploration, preprocessing, feature engineering, clustering, dimensionality reduction, and anomaly detection
2. **Machine Learning** - Model training, evaluation, and comparison
3. **Optimization** - Hyperparameter optimization using Grid Search and Random Search

## Features

- **Bilingual Support**: All notebooks available in both English (EN) and Turkish (TR)
- **Complete Pipeline**: From raw data to optimized models
- **Comprehensive Analysis**: EDA, preprocessing, feature engineering, and model evaluation
- **Multiple Algorithms**: Various ML models and optimization techniques
- **Saved Models**: Trained models saved for easy deployment

## Requirements

- Python 3.8+
- Jupyter Notebook
- Required packages: pandas, numpy, scikit-learn, matplotlib, seaborn, joblib

## Usage

1. Navigate to the desired project directory
2. Open the notebooks in Jupyter
3. Run cells sequentially to execute the analysis

## Data Pipeline

```
Data Mining Project → Machine Learning Project → Optimization Project
     ↓                        ↓                          ↓
Raw Data → Preprocessed → Engineered Features → Optimized Models
```
